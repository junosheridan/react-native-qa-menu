# react-native-qa-menu

Helper menu for QAs to examine the apps during the development

## Installation

```sh
npm install react-native-qa-menu
```

## Usage

```js
import { QaMenuView } from "react-native-qa-menu";

// ...

<QaMenuView color="tomato" />
```
